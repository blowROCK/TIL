

## [ euc-kr ] 

```
ascii 코드 + 한글 
영문/숫자/기호는 1바이트. 한글과 한자는 2바이트 
euc-kr 또는 ksc5601 
웹페이지 작성에 사용가능 
특수한 외국어 문자나 일본식/중국식 한자는 표현불가 
```

## [ unicode ] 
```
모든 글자를 2바이트로 표현 
전세계 모든 글자들을 한꺼번에 표현가능 
웹페이지 작성에 사용불가 
 ```
 
## [ utf-8 ] 
```
영문/숫자/기호는 1바이트로, 한글과 한자 등은 3바이트로 표현 
전세계 모든 글자들을 한꺼번에 표현가능 
웹페이지 작성에 사용가능
```
